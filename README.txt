HE THONG QUAN LY CONG VIEC DU AN
================================

Mon hoc:
Ky thuat lap trinh

De tai:
Xay dung he thong quan ly cong viec du an bang ngon ngu C


1. Gioi thieu
--------------
Chuong trinh quan ly thong tin du an, thanh vien tham gia,
cong viec va thoi gian thuc hien.

Kien thuc ap dung:
- Struct trong C
- Con tro
- Danh sach lien ket
- Doc ghi file


2. Cau truc thu muc
-------------------

source/
    qlda.c

data/
    duan.txt
    thanhvien.txt
    timesheet.txt


3. Mo ta file du lieu
---------------------

duan.txt:
Luu thong tin cac du an.

thanhvien.txt:
Luu thong tin thanh vien tham gia du an.

timesheet.txt:
Luu thong tin cong viec va thoi gian thuc hien.


4. Cach chay chuong trinh
-------------------------

- Mo file qlda.c bang trinh bien dich C.
- Bien dich chuong trinh.
- Dam bao cac file du lieu nam dung thu muc.
- Chay chuong trinh.


5. Chuc nang chinh
------------------

- Quan ly du an:
  + Them, hien thi, tim kiem, cap nhat, xoa.

- Quan ly thanh vien:
  + Them thanh vien, gan vao du an, cap nhat trang thai.

- Quan ly cong viec:
  + Them timesheet, theo doi trang thai.

- Bao cao:
  + Thong ke gio lam va chi phi.


6. Huong phat trien
-------------------

- Them phan quyen nguoi dung.
- Ket noi co so du lieu.
- Xay dung giao dien truc quan.
